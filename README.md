# 🌦️ Climate Chase

Climate Chase is a modern web-based weather application that provides real-time updates on weather conditions.
Stay informed, plan your day better, and stay ahead of changing weather conditions with ease.

![ClimateChaseDemo](https://github.com/KrishBharadwaj5678/ClimateChase/raw/main/ClimateChaseDemo.png)

## ✨ Features

| Feature                 | Description                                        |
| ----------------------- | -------------------------------------------------- |
| 🌍 Global Weather       | Get weather updates for any city worldwide         |
| 🌡️ Real-Time Data      | View current temperature, humidity, and conditions |
| ⛅ Forecast              | Access upcoming weather predictions                |
| 🔍 Search Functionality | Easily search weather by city name                 |
| 🌐 Web-Based            | Works on any browser and device                    |

---

## 🛠️ Tech Stack

| Technology             | Purpose                             |
| ---------------------- | ----------------------------------- |
| 🌐 **HTML5**           | Structure of the web application    |
| 🎨 **CSS3**            | Styling and responsive design       |
| ⚡ **JavaScript**       | Adds interactivity and API handling |
| ☁️ **OpenWeather API** | Fetches real-time weather data      |

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/KrishBharadwaj5678/ClimateChase.git
```

### 2️⃣ Navigate to the Project

```bash
cd ClimateChase
```

### 3️⃣ Open in Browser

Simply open `index.html` in your browser.
